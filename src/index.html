Welcome to Spacetime Klein Bottle Particle Modeler

Understanding Particle Physics

An interactive tool for modeling fundamental particles using the Spacetime Klein Bottle (SKB) framework.

What is SKB?

A topological model using twists and links to describe particle properties.

Capabilities

  • Model quarks and composite particles
  • Analyze charge and mass
  • Validate configurations

Key Concepts

  • Twist Numbers: Affect charge
  • Linking Pairs: Influence mass
  • Mass Parameters: Tune predictions
